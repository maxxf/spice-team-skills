## Sequence Logic

### Trigger Rules

```python
def get_sequence(score: int) -> str:
    if score < 50:
        return "POOR"
    elif score < 70:
        return "FAIR"
    elif score < 85:
        return "GOOD"
    else:
        return "EXCELLENT"
```

### Personalization Variables

| Variable | Source |
|----------|--------|
| `{brand}` | Audit input |
| `{score}` | Audit output |
| `{first_name}` | CRM / form |
| `{market}` | Audit input |
| `{gap_1}`, `{gap_2}`, `{gap_3}` | Top 3 scoring gaps |
| `{strength_1}`, `{strength_2}` | Top 2 scoring strengths |
| `{impact_X}` | Estimated % lift per gap |
| `{competitor}` | Top local competitor |
| `{projected_monthly_gain}` | ROI calculation |
| `{hero_issue}` | Hero image specific finding |
| `{menu_count}` | Category count |
| `{rating}` | Platform rating |
